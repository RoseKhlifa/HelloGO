# 大一 → 大二下实习 路线规划

> 起点：2026/5/11，大一下学期还剩约 2 个月
> 终点：2027 年暑期拿到第一份实习
> 主线：Go + Java 后端 + TypeScript/React 前端

---

## 阶段 0：现在 ~ 2026 年 7 月（大一下尾巴，2 个月）

**目标：把 Go 写顺，建立"我能做出东西"的肌肉记忆。**

### 学习

- **Go**：用 2 周走完 *A Tour of Go* + *Go by Example*，然后立刻开始写代码。不要看完整本书再动手。
- **配套**：
  - Git 熟练度（不只是 add commit push，要会 rebase、cherry-pick、解决冲突）
  - Linux 常用命令
  - VSCode 或 GoLand 配置
- **算法**：LeetCode 开账号，每天 1-2 题 Easy，目标月底前 30 题。用 Go 写。
- **学校的课**：C/C++、离散数学、高数——**别糊弄**，特别是离散和数据结构，这是后面所有东西的基础。

### 项目 #1（必做）

一个 Go 写的 CLI 工具，候选方向：

- 从命令行查询 B 站视频信息的小工具
- 简易 markdown 笔记管理器
- GitHub trending 抓取器

要求：

- 开源到 GitHub
- 写正经的 README
- 加 GitHub Actions 跑测试

这是你的第一个"门面"。

### 别做的事

- 别学 Java
- 别学 Docker/K8s
- 别学微服务
- 别背八股

现在背等于白背。

---

## 阶段 1：2026 年 7 月 ~ 9 月（大一暑假，2 个月）

**目标：变成一个会做全栈项目的人。**

### 学习

- **TypeScript + React + Next.js**：3 周
  - 资源：官方教程 + *Next.js Learn* 官方课程
  - 配合：Tailwind + shadcn/ui
- **PostgreSQL**：1 周
  - 学会 SQL + 设计表 + 用 ORM
  - Go 用 sqlc 或 gorm
  - TS 用 Prisma
- **部署**：买一台便宜 VPS（搬瓦工/雨云 ¥30/月）
  - 学会用 Docker + docker-compose 部署
  - 加 Nginx 反代和 HTTPS
- **算法**：暑假 2 个月每天 2 题，9 月开学时累计 90~120 题。

### 项目 #2（关键）

一个真正的全栈项目。**选你自己会用的东西**，不要做电商/博客/Todo。候选方向：

- 追番/追剧记录站
- 针对你专业课的题库刷题站
- 校内二手书交换平台
- RSS 阅读器

要求：

- Next.js 前端 + Go 后端 + PostgreSQL
- 用户系统（注册登录）
- 部署到 VPS
- 真正能访问的域名

## 阶段 2：2026 年 9 月 ~ 2027 年 1 月（大二上，4 个月）

**目标：Java 启动 + 项目深度 + 算法持续。**

### 学习

- **Java**（Sept-Oct，1.5 个月）：
  - Java 语法 + 集合 + 多线程 + JVM 基础
  - 资源：*Java 核心技术卷一* + JavaGuide 网站
- **Spring Boot**（Nov-Dec）：
  - **直接用 Spring 把项目 #2 的后端重写一遍**
  - 这是最高效的学法，因为你已经用 Go 写过同样的东西，会瞬间明白 Spring 在抽象什么
- **算法**：每天 2 题
  - 刷完代码随想录一遍
  - LeetCode hot 100 一遍
  - 期末前累计 200 题
- **课程重点**：**操作系统、计算机网络、数据库系统**
  - 三门是面试必问
  - 跟紧学校的课，配合 *小林 coding* 食用

### 项目 #3（含金量）

二选一：

**A 路线（深度）**：用 Go 自己撸一个 mini 项目

- 简易 Redis、HTTP 框架、KV 存储
- 可以参考 *7days-golang* 系列
- 简历上写"从零实现 XX"非常加分

**B 路线（开源）**：

- 找一个 star 数 1k-10k 的 Go/TS 开源项目
- 从 good first issue 入手
- 提 1-2 个有意义的 PR 被 merge
- **这条对双非含金量极高，比任何自己写的项目都管用**

## 阶段 3：2027 年 2 月 ~ 4 月（大二下前半，3 个月）

**目标：实习突击期，进入面试态。**

- **八股密集**：JavaGuide + 小林 coding，每天 1 个模块
  - 重点：JVM、并发、MySQL、Redis、Spring、计网、操作系统
- **算法**：LeetCode hot 100 二刷 + 剑指 offer
  - 4 月底累计 300+ 题
- **项目复盘**：
  - 把 2-3 个项目的"技术亮点"提炼成简历语言
  - 每个项目准备 5 个能答 10 分钟的深入问题（为什么这么设计、遇到什么问题、怎么解决的）
- **简历精修**：找学长学姐改、找牛客论坛上的老哥改，至少改 5 轮
- **模拟面试**：找同学互面、用 Pramp 或牛客模拟面试

---

## 阶段 4：2027 年 4 月 ~ 7 月（大二下后半 + 暑假入口，3 个月）

**目标：投递 + 面试 + 拿 offer。**

### 投递策略（双非大二，要海投 + 内推）

- **第一梯队尝试**：字节、腾讯、阿里、美团、shopee（有大二实习机会，但难）
- **第二梯队主攻**：B 站、滴滴、小红书、知乎、PingCAP、得物
- **保底**：中小厂、外企（思科、Intel 北京）、远程实习

渠道：BOSS 直聘 + 内推（关键，混学长群、技术社区、Discord/Twitter）

### Go 岗位 vs Java 岗位

双线投：

- **Go 岗位**：竞争小、和你能力匹配度高（字节、B 站、PingCAP、滴滴部分团队）
- **Java 岗位**：多但卷

### 拿到 offer 后

- 暑假实习 2-3 个月
- 转正机会不大，但留下"我有大厂实习"标签巨大

## 全程通用建议（不分阶段）

1. **GitHub 当简历用**
   - 从今天开始，所有代码上 GitHub
   - commit 信息认真写
   - 每天看看绿格子

2. **技术博客**
   - 大二开始写
   - 不要写"XX 入门教程"——写"我做 XX 项目时踩的坑"
   - 一年 20 篇就够，面试时是巨大加分

3. **英语**
   - 每天读 30 分钟英文技术文档（Go 官方博客、Vercel/Next.js 博客）
   - 双非补差距的隐形武器
   - 外企远程实习要求英语

4. **不要碰的坑**
   - 不要去培训班
   - 不要刷大学生计算机设计大赛（含金量低）
   - 不要陷入"再学一门语言/框架"循环
   - 不要相信"30 天精通 XX"
   - 不要做电商/博客/Todo 项目当门面
